Harvey's Degree

My degree is in Computer Science from Brunel University London.

During the course of the degree, a large number of modules were undertaken and several of these involved substantial coursework projects. Details of the modules taken and their respective projects are presented below.

Year 1

Modules

  • Level 1 Group Project
  • Software Design
  • Software Implementation
  • Fundamental Programming Assessment
  • Data and Information
  • Introductory Programming
  • Logic and Computation

Coursework and Projects

  • Various Java programming projects to build foundational knowledge
  • A project conducted to extract Twitter analytical data in order to perform data analysis upon, primarily using SQL commands
  • A group project conducted to control and manoeuvre a Finch Robot to showcase our programming knowledge. The Finch was required to take in various commands (in the format of a Stack) and execute those commands that were given in a LIFO order. The project was coordinated with 4 other students and required a demonstration in front of the module leaders.

Year 2

Modules

  • Level 2 Group Project
  • Software Development and Management
  • Usability Engineering
  • Algorithms and their Applications
  • Networks and Operating Systems

Coursework and Projects

  • Various coding projects in Java utilising a wide range of key programming areas such as IO, Threading, Parsing, Database Interaction and SQL, Java servlets and sockets
  • UML modelling and software development methodologies
  • Level 2 group project developing an Android mobile application of our choice. Our project was CrazySpoon, an app that allowed users to find and order food from restaurants within their local area.
  • A usability evaluation of the app created during the Level 2 group project. The report showcased the reasons and background behind the UI and UX decided upon using relevant usability evalution methods, and relevant usability engineering concerns. The coursework was presented in front of the module leaders.

Year 3

Modules

  • Final Year Project
  • Advanced Topics in Computer Science
  • Software Project Management
  • Network Computing
  • Digital Media and Games

Coursework and Projects

  • A year-long development project to create and train an Artificial Neural Network (ANN) for the purpose of detecting network intrusions using packet analysis. The ANN was trained to detect ~87% of intrusions using a test set.
  • A project to develop a game using the MonoGame development environment. The game prototype was a "shoot-em-up" based in space and had two levels of incremental difficulty, with the third and final level being a "boss" battle.
  • A project to develop a banking system, implemented using multithreading and parallelism. The project was written in C# and was very well received.
